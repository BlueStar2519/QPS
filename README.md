# Quiet Presence Scan Wizard

A Next.js application for the Quiet Presence Scan (QPS) wizard - a guided, card-by-card scan of brand presence.

## Getting Started

First, install the dependencies:

```bash
npm install
```

Then, run the development server:

```bash
npm run dev
```

Open [http://localhost:3000](http://localhost:3000) with your browser to see the result.

## Features

- Multi-step wizard interface
- Role-based answering (owner vs clients)
- Five Quiet Presence pillars
- Global Brand Health Indicators (GHI) guide
- Summary views with comparisons
- PDF generation placeholders
